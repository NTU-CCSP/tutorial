Box Model

CSS basic box model

Box & display 屬性

  • 文字與元素排版
  • 蓋成長方形 box
  • box 有兩種
  • CSS 決定大小和位置

Block-level boxes

display: block | list-item | table | table-*

Inline-level Boxes

display: inline | inline-block

細說 display

  • none: 連 box 都沒有。
  • inline: 一秒變 inline element
  • block: 一秒變 block element
  • inline-block: 當成 inline element 來排版（左到右）
    但又能設 width, height 等 inline element 不能用的屬性。
  • table 系列：當成 table 來排版，見標準。

margin, border, padding

心室肥大

注意 .heart 的寬度與高度

margin 二三事

  • 從 border 向外算，和其他元件的距離
  • inline 元素的 margin-top、margin-bottom 會被忽略
  • margin-left 和 margin-right 同為 auto 時，元素會置中。

margin 二三事

  • block 元素的 margin-top, margin-bottom 密接時，有時只會表現較大的那一個 （Collapsing margin）
  • margin-top 可能會超出父元素。

border 與 padding

  • border 會佔有空間、推開周圍元素；outline 則不會。
  • padding 是 border 與內文的間隙。
  • 元素的背景顏色會填滿 border 內的空間
  • 背景顏色的寬 = 元素 width + 左右之 padding；高亦如是。

切換 box model

box-sizing: border-box; (瀏覽器支援)

width = border + padding + content

Z-index

圖層疊疊樂

z:index: 整數

把元素變成「一層」

  • 類似 Photoshop 的群組（資料夾）
  • 也像大廈裡的其中一層樓
  • Spec: 「Stacking Context」
  • 群組內可另外設 z-index（樓中樓）

萬丈高樓平地起

蓋大樓：

  • z-index： 樓層號碼。
  • stacking context：地基。

形成 stacking context：

  • position: 非 static
  • z-index != 0 或 opacity < 1

蓋房子順序：

地基 → 地下室 → 低樓層 → 高樓層

  1. stacking context 之背景與邊界
  2. 負 z-index 的 element
  3. 地表：z-index:0 | auto 的 element
  4. 正 z-index 的 element

同一層裡面

z-index 相同的元素，用下面規則分高下。

（由低至高）

  1. Normal flow 中的 block elements
  2. float elements
  3. 文字等 inline elements
  4. position != static 的 element (Positioned elements)

若兩元素同一層，則先寫在 HTML 檔裡的，會被壓在下面。

Stacking Order

Positioned 元素 > Inline 元素 > float 元素 > block in Normal Flow
