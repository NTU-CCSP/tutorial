HTML

"Markup" : 標籤

替文字標上「語意」

每個人都應該努力培養雅量。

<p> 標出「段落」

<strong> 標出「重點」

HTML：勾勒內容與結構的 "Formatting Language"

有些標籤有特異功能

Tag, Element, Attribute

標籤、元素、屬性

  • <label for="chk-tos">：
    　有著 for attribute 的 start tag。
  • </label>：
    　end tag。
  • <label for="chk-tos">我已……條款</label>：
    　label element。
  • 有些 element 僅由單一 tag 構成，如 <input>, <hr>, <img>等。
  • XHTML → <hr />

<br> 與 &nbsp;

用 <br> 換行，空白鍵預設會重疊

&nbsp; 代表「空白 character entity」，可疊加

<!-- --> 是多行註解

Some Tags & Their Meanings from Google

MDN HTML Elements

語意標籤

<h1>~<h6>：標題。
<p>：段落； <blockquote>：引文。
<strong>：重點； <em>：強調。

語意標籤

<hr>：分隔線。
<ol>,<ul>：有序/無序列表；<li>：列表的項目。
<dl>：定義列表； <dt>：被定義的項目； <dd>：定義。

表格

<table> 表格（table）； <tr> 表格列（table row）；
<td> 資料格子（table data cell）； <th> 表頭格子（table header）。

tr 裡只能裝 td 或 th；td裡啥都能裝。

標籤神秘力量

<img> 圖, <a> 超連結, <iframe> 嵌入框

<img src="網址" alt="圖壞掉時顯示的字" (width="寬" height="高")>
<a href="網址" (target="_blank")>顯示的文字</a>
<iframe src="網址"></iframe>

表單

表單

表單內若含有 <input type="file">，
請在 <form> 加入 enctype="multipart/form-data"

勾勾和點點

name 屬性一樣 → 同一組

<label> 的 for 屬性和 <input> 的 id 屬性配對

無語意標籤

  • 不知道要用什麼包，但還是想包在一起
  • <div>, <span>
  • 常常搭配 CSS 套樣式、或當做 javascript 的操作對象。

HTML5 語意標籤

<section>：章節 / 小節
<article>：一篇文章
<section> 可以包 <article>， <article> 也可以包 <section>。

<nav>：導覽選單
<header>,<footer>：文章段落或整個頁面的開頭、結尾

通用屬性

大部分 element 都有下面屬性：

  • class,id：替該 element 加上一個 classname / unique ID，使 CSS 的 selector 能選取 / 讓 javascript 方便操作。
  • title：工具提示（tooltip）
  • style：加上 CSS Declaration。勿和 <style> 標籤混淆。
  • tabindex：決定可 focus 元素的 tab 順序。
  • contenteditable：可任意編輯元素內容。

HTML 文件架構

  • <head> 存放「不是網頁內文」的其他資訊。

  • <body> 放網頁內容。
  • <html> 包住 <head> 和 <body>。
