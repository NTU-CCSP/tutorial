陣列實字 Array Literal
-------

```js
var shoppingList = ["布料", "棋盤", "稿紙", "綠豆糕"];
```

以上陣列實字，和下面等價

```js
var shoppingList = [];
shoppingList.push("布料");
shoppingList.push("棋盤");
shoppingList.push("稿紙");
shoppingList.push("綠豆糕");
```

---

操作陣列
-----------

若有個字串變數名叫 `list`……

* `list[位置]`：取得此位置的元素（位置從 0 起算）
* `list.length`：長度
* `list.indexOf(元素)`：元素在陣列的位置（從左邊數來）
* `list.slice(位置)`：切陣列
