`String` 字串
------------

`count = count + 1;` 是給電腦執行的程式碼。

----

給人來讀的「句子」要用引號 `" "` 或 `' '` 括起來，

告訴瀏覽器「這段句子不是程式碼，是給人看的文字」。

```javascript
var count = 3;
console.log("我買" + count + "個橘子去，你就在此處不要走動。");
// 我買3個橘子去，你就在此處不要走動。
```

操作字串
------

若有個字串變數名叫 `str`……

* `str[位置]`：取得此位置的字（位置從 0 起算）
* `str.length`：長度
* `str.indexOf(字)`：字在字串的位置（從左邊數來）
* `str.slice(位置)`：切字串
